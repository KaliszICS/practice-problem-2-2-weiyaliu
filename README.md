# Instructions  

1. Create a method called isEven that accepts an integer as a parameter.
Return true if it is even, false if it is not. (0 will be considered even).


2. Create a method called isOdd that accepts an integer as a parameter.
Return true if it is odd, false if it is not. (0 will be considered even).


3. Create a method called isPositive that accepts an integer as a parameter.
Return true if it is positive, false if it is not.


4. Create a method called isNegative that accepts an integer as a parameter.
Return true if it is negative, false if it is not.

5. Create a method called combinedLength that accepts two Strings as paramters.
Return the length of the two strings combined.
